## ✨ 功能特性

- 📊 实时内存监测（每2秒检测）
- 🟡 分级警告（75%黄 / 85%橙 / 95%红）
- 📺 HUD 顶部警告条
- 💬 聊天框警告
- 💾 紧急自动保存
- 🚪 强制退出保护

## 🛠️ 安装方法

1. 安装 [Fabric Loader](https://fabricmc.net/use/)
2. 下载对应版本的 [Fabric API](https://modrinth.com/mod/fabric-api)
3. 将对应版本的 `.jar` 文件放入 `.minecraft/mods` 文件夹
4. 启动游戏

## ⚙️ 配置文件

首次运行后在 `config/xiaozi_core.json` 生成配置。
